# Security Policy

## Supported Versions

Only the latest version of this extension is currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please follow these steps:

1.  **Do not disable the issue publicly.**
2.  Email the project maintainer directly or open a draft security advisory in GitHub if enabled.
3.  Include detailed steps to reproduce the vulnerability.
4.  We will respond as quickly as possible to acknowledge your report.
5.  We will verify the fix and release a patch.

Thank you for helping keep this project safe!
