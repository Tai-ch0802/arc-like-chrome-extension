使用指南

從安裝到進階操控，一步步掌握側邊欄的每一項功能。

快速開始

安裝擴充功能

  1. 前往 Chrome 線上應用程式商店，點擊「加到 Chrome」。
  2. 安裝完成後，點擊 Chrome 工具列右上角的擴充功能圖示（拼圖圖示），找到「Arc-like Sidebar」並釘選。
  3. 點擊圖示或使用快捷鍵 Cmd/Ctrl + I 開啟側邊欄。

小提示：首次開啟側邊欄後，按任意方向鍵即可自動取得焦點並開始鍵盤導覽。

垂直分頁管理

側邊欄會垂直顯示所有當前視窗的分頁，讓你一目了然地看到完整標題，不再像傳統水平標籤列那樣被壓縮成小圖示。

  • 切換分頁：點擊任意分頁即可跳轉。
  • 關閉分頁：將滑鼠停在分頁上，右側會出現關閉按鈕。
  • 拖曳排序：長按並拖曳分頁來自由調整順序。
  • 建立新群組：將滑鼠停在分頁上，右側會出現以該分頁建立新群組按鈕。
  • 加入書籤：可選擇拖曳至書籤資料夾上，或是將滑鼠停在分頁上，右側會出現加入書籤按鈕。
  • 右鍵選單：在分頁上按右鍵可存取更多操作，包括「複製網址」、「加入閱讀清單」等。

分頁群組

完整整合 Chrome 原生分頁群組功能。群組的名稱與顏色會即時同步呈現。

  • 展開/收合：點擊群組標題即可切換展開或收合狀態。
  • 拖曳分頁進群組：將分頁直接拖入群組上方即可加入。

書籤管理

側邊欄內建完整的書籤瀏覽器，直接顯示你的書籤資料夾結構，支援展開、搜尋與拖曳操作。

  • 瀏覽書籤：以樹狀結構展示所有書籤資料夾，點擊展開子資料夾。
  • 開啟書籤：點擊書籤即開啟新分頁，並自動建立 Linked Tab 關聯。
  • 拖曳變書籤：將分頁拖曳到書籤區域，它會立即被儲存為書籤。
  • 智慧渲染：使用動態渲染技術，即使有上千筆書籤也能流暢滾動。

Linked Tabs 連結分頁

這是我們最獨特的功能！當你從側邊欄開啟書籤時，系統會自動建立書籤與分頁之間的「連結」，幫助你避免重複開啟，並即時同步狀態。

運作方式

  • 自動建立：點擊書籤開啟分頁時自動記錄關聯；將分頁拖曳成為書籤時也會自動關聯。
  • 視覺指示：已連結的書籤旁會顯示連結圖示，讓你一眼辨識哪些已開啟。
  • 查看面板：點擊連結圖示可展開面板，顯示所有已開啟的相關分頁，支援快速跳轉。
  • 自動清理：分頁被關閉、URL 改變或書籤被刪除時，關聯會自動移除。

連結狀態會透過 Local Storage 持久保存，即使重新啟動瀏覽器也不會遺失。

跨視窗管理

不只是當前視窗！側邊欄會在分頁列表下方顯示「其他視窗」區塊，讓你掌握所有 Chrome 視窗的分頁狀態。

  • 視窗總覽：列出除目前視窗外的所有視窗與分頁，包含群組資訊。
  • 快速跳轉：點擊目標分頁，Chrome 會自動切換至該視窗並聚焦分頁。
  • 即時更新：其他視窗的分頁變動會自動反映在側邊欄中。
  • 唯讀安全：此區塊設計為唯讀模式，避免誤操作影響其他視窗的結構。
  • 視窗命名：為不同的 Chrome 視窗設定自訂名稱（例如「工作」、「個人」），讓多視窗管理更加一目了然。

專業級搜尋

側邊欄頂部的搜尋框提供強隊的即時過濾功能，讓你在數十個分頁與上千筆書籤中瞬間鎖定目標。

  • 多關鍵字：支援空格分隔的多關鍵字過濾（例如 google docs work），多詞精確搜尋。
  • Domain 搜尋：輸入網域（如 github.com）即可篩選出該網站的所有分頁與書籤。
  • 即時高亮：搜尋結果會即時標記匹配的關鍵字，保持視覺焦點清晰。
  • 跨視窗：搜尋結果包含所有 Chrome 視窗的分頁，不限於當前視窗。

閱讀清單

側邊欄將 Chrome 原生閱讀清單深度整合，讓你無需切換面板即可管理「稍後閱讀」的文章。

基本操作

  • 加入文章：在分頁右鍵選單中選擇「加入到閱讀清單」。如果 URL 已存在，選項會自動禁用。
  • 開啟閱讀：點擊清單項目會在新分頁中開啟，並自動標記為已讀。開啟的分頁會自動歸入「來自閱讀清單」群組。
  • 標記狀態：切換已讀/未讀狀態，方便追蹤閱讀進度。已讀項目會以較淡的樣式顯示。
  • 過期提醒：超過 3 天的已讀項目會顯示「已讀 X 天前」標籤，提醒你清理不再需要的文章。
  • 即時同步：與 Chrome 原生閱讀清單雙向同步，任何變動即時反映在 UI 上。

RSS 訂閱

不用額外安裝閱讀器！在設定面板中訂閱你感興趣的網站 RSS Feed，系統會自動排程抓取最新文章並加入閱讀清單。

管理訂閱

  • 新增訂閱：在側邊欄設定 → RSS 區塊中輸入 Feed URL 即可新增。
  • 更新間隔：每個訂閱可獨立設定抓取間隔，選項包含 1 小時、3 小時、8 小時、12 小時與 24 小時。
  • 暫停/恢復：可暫停特定訂閱的自動抓取，不影響其他訂閱。
  • 智慧去重：使用 hash 比對機制，確保同一篇文章不會重複加入閱讀清單。
  • 手動抓取：支援立即觸發抓取，不必等待排程。

AI 智慧整理

透過 Chrome 內建的 Gemini Nano 本機 AI，一鍵將散亂的分頁依語意自動分類成群組。所有處理 100% 在你的設備上進行，資料絕不上傳雲端。

使用方式

  1. 確保你已完成 AI 功能設定（需 Chrome 140+、22GB 磁碟空間）。
  2. 點擊側邊欄中的「✨ 智慧整理」按鈕。
  3. 系統會掃描所有未分類的分頁（需至少 4 個），AI 將在數秒內完成分析。
  4. 分頁會自動被移入帶有 Emoji 與描述性名稱的群組中。

不滿意分類結果？別擔心！整理完成後會顯示「復原 (Undo)」按鈕，點擊即可還原到整理前的狀態。

外觀客製化

側邊欄提供畫布級的視覺客製化能力，從預設主題到完全自訂，打造你獨一無二的工具。

預設主題

內建多種精美主題（Geek、Google、Darcula、Geek-Blue、Christmas 等），在設定中一鍵切換。

自訂配色

選擇「自訂 (Custom)」主題後，可透過色彩選擇器自由設定三大核心色——主背景色、強調色與文字色。系統會自動透過 HSL 演算法衍生出和諧的輔助色票，確保 WCAG 對比度達標。

背景圖片

  • 上傳圖片：支援 JPG、PNG、WebP 格式，自動壓縮為 WebP（最大 1920px）。
  • 圖片網址：直接貼上圖片 URL，系統會自動抓取並快取。
  • 微調控制：提供透明度 (0–100%)、模糊度 (0–20px)、水平/垂直位置對齊的精細調整。

社群分享

將你的自訂配色匯出為 JSON 檔案分享給朋友，也可以載入別人分享的主題，快速套用社群熱門配色。

鍵盤導覽與快捷鍵

側邊欄支援完整的鍵盤操作體驗，讓進階使用者無需滑鼠也能完成所有操作。

全域快捷鍵

快捷鍵 功能
Cmd/Ctrl + I 開啟/關閉側邊欄
Alt/Opt + T 在當前分頁旁新建分頁

側邊欄內導覽

快捷鍵 功能
↑ / ↓ 在分頁、群組、書籤項目間移動焦點
← / → 進入/離開項目的操作按鈕（關閉、編輯等）
Enter / Space 執行主要動作（切換分頁、展開資料夾）
F2 重新命名書籤
Delete 刪除選中的項目

焦點小技巧：當焦點在列表最頂部時，按 ↑ 會自動跳至搜尋框；在搜尋框中按 ↓ 即回到列表首項。
