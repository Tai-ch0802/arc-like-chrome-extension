啟用本機 AI 功能

只需三個步驟，即可在你的裝置上解鎖 AI 智慧整理的強大能力。

系統需求

Chrome 140 或更新版本
≥ 22 GB 可用磁碟空間
GPU VRAM > 4GB 或 RAM ≥ 16GB
Windows 10+, macOS 13+, Linux
1

啟用本機模型

在 Chrome 網址列輸入 chrome://flags/#optimization-guide-on-device-model，將選項設為「已啟用」。

2

啟用 Prompt API

同樣在網址列輸入 chrome://flags/#prompt-api-for-gemini-nano，設為「已啟用」或「已啟用多國語言」。

3

重新啟動並驗證

重新啟動 Chrome，開啟 DevTools Console 輸入 await LanguageModel.availability(); 確認回傳 "available"。

官方文件列出支援英文、西班牙文與日文，但實際上其他語言也能正常使用。若你使用的語言不在官方清單中，仍然歡迎嘗試！
