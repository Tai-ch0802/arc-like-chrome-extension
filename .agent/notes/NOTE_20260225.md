## 技術決策
- **動態版本號**：為了避免每次 Release 都需要手動修改官網，透過前端直接 Fetch GitHub API，讓網站可以保持在最新狀態。加入了 `sessionStorage` 進行輕量級快取，並處理了速率限制 (Rate limit) 下的回退機制 (Fallback to hardcoded version)。
- **純 CSS AI 群組化動畫**：拒絕使用 GIF 或影片，而是選用純 CSS 與 DOM 實作，確保動畫本身是清晰的向量呈現，並且 DOM 上的文字都可以完美的掛載 i18n 翻譯 `data-i18n`，讓所有語系的使用者都能看到在地化語言的介面模擬動畫。
- **Clip-Path 疊加技法**：在 AI 動畫中，將兩組完全相同的佈局清單疊在同一個位置（`absolute`），透過操縱 `clip-path` 切割遮罩的可見範圍（`inset`），達到如同雷射區塊擦除並顯示下層內容的效果。
