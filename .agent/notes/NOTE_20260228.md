## 技術決策
- **Translation Automation**: 面對 14 個語系的翻譯擴充，選擇使用 Python 腳本 (`locales/i18n_updater.py`) 自動化塞入英、中字串。不僅減少重複勞動，也避免語系檔格式被意外破壞，並維持前端切換語系的穩定性。
- **Focus on Mindset Shift**: Feature page 不去比長長的「功能清單」，而是選擇比「設計理念 (Philosophy)」：主打不需設定、Arc-like 美學與無痕整合。
