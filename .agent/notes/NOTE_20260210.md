# 開發摘要 - 2026-02-10 ~ 2026-02-11

## 變更項目
- [modules/rssManager.js]: 修復 RSS 訂閱新增失敗。新增 `parseRssFeedXml()` 本地解析函數，修改 `fetchRssFeed()` 偵測執行環境：side panel 本地 fetch+parse，service worker 委派至 offscreen document。
- [offscreen.js]: 新增 `fetchAndParseRssFeed` message action，讓 offscreen document 同時處理 HTTP 抓取與 XML 解析。
- [background.js]: **關鍵修復** — `onMessage` listener 對已處理的 action return `false`，對不認識的 action 不做任何攔截，避免關閉 offscreen document 的回應通道。
- [Makefile]: 在 `PROD_STATIC_FILES` 中加入 `offscreen.html` 和 `offscreen.js`，修復 production build 缺少 offscreen 檔案的問題。
- [PRIVACY_POLICY.md]: 更新隱私權政策。

## 技術決策
- **根因**：`chrome.runtime.sendMessage()` 會廣播到所有 context，包含 service worker 自己。background.js 的 `onMessage` listener 收到 `fetchAndParseRssFeed` 訊息但不處理也不 return `true`，導致 Chrome 關閉通道，offscreen document 的回應無法送回。
- **修復**：確保 background.js 的 listener 對不認識的 action 不做任何攔截。
