## 🔥 主な機能

### 🔗 独自の革新：リンクされたタブ (Linked Tabs)
これが私たちの最も強力な機能です！サイドバーからブックマークを開くと、自動的に**「リンク」**が作成されます。
- **タブの乱雑さを回避**: ブックマークの横にあるリンクアイコンをクリックすると、そのブックマークから開かれたすべてのタブが表示され、重複して開くのを防ぎ、システムリソースを節約できます。
- **双方向同期**：タブを閉じると、ブックマークの状態が自動的に更新されます。ブックマークを削除すると、リンクされたタブはインテリジェントに処理されます。
- **視覚的フィードバック**：ブックマークの横に洗練されたリンクアイコンが表示され、どれが現在アクティブかが一目でわかります。

### ⚡️ 爆速パフォーマンス：仮想スクロール
数千のブックマークがありますか？問題ありません！
- **ゼロレイテンシ**：**10,000+** のブックマークがあっても、サイドバーはシルクのように滑らかなままです。
- **メモリ最適化**：表示されているアイテムのみをレンダリングする高度な仮想化エンジンを搭載しており、メモリ使用量を大幅に削減します。

### 🔍 プロ仕様の検索
ただ探すのではなく、瞬時に見つけます。
- **マルチキーワードフィルタリング**：スペース区切りのキーワード（例：「google docs 仕事」）をサポートし、正確なターゲティングを可能にします。
- **ドメイン検索**：ドメイン（`github.com`など）を入力すると、特定のソースからのタブやブックマークを瞬時にフィルタリングできます。
- **スマートハイライト**：一致したキーワードをリアルタイムでハイライトし、視覚的な焦点を明確に保ちます。

### 🗂️ 統合ワークスペース
- **垂直タブ**：ページタイトル全体を表示し、小さなアイコンに圧縮されることはもうありません。
- **ネイティブグループサポート**：Chromeタブグループと完全に統合され、色と名前が同期されます。
- **ドラッグ＆ドロップ**：直感的な管理—タブ、グループ、ブックマークフォルダ間でアイテムを簡単に移動できます。
- **ドラッグして保存**：タブをブックマークエリアにドラッグして瞬時に保存。ブックマークをタブエリアにドラッグして開きます。

### 🎨 プレミアムデザイン
- **フォーカスモード**：目の疲れを軽減するために慎重に調整されたコントラストを持つ、洗練されたダークテーマ。
- **自動展開**：アイテムをドラッグしながらフォルダの上にホバーすると、パスが自動的に展開されます。
- **スマートホバー**：アクションボタンは必要なときだけ表示され、インターフェースをクリーンで集中できる状態に保ちます。

## ⌨️ 生産性ショートカット
- **Cmd/Ctrl + I**：サイドバーの切り替え
- **Opt/Alt + T**：現在のタブの隣に新しいタブを作成
