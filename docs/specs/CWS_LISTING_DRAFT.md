# CWS Listing Draft: Arc-like Sidebar

## 1. Short Description (Max 132 chars)
「Arc-like Sidebar」你的瀏覽器，理當井然有序。結合垂直分頁、書籤整合與 100% 隱私的本機 AI 智慧整理，打造最優雅的資訊指揮中心。

*English (130 chars)*
Arc-like Sidebar: Vertical tabs, bookmarks, and a 100% private local AI organizer. Transform your browser into an elegant command center.

## 2. Detailed Description

**【不是更多設定，而是更好的瀏覽體驗】**
受夠了擁有超過 100 個開關、眼花撩亂的分頁管理工具嗎？
「Arc-like Sidebar」採用原生介面整合與 Opinionated（固執的極簡美學）設計，為您帶來開箱即用、絲滑順暢的右側/左側邊欄。不只找回乾淨的頁面，更透過 Chrome 內建的本機 AI，讓資料整理變得前所未有的簡單。

**🚀 關鍵差異化特色：**

**1. 🔒 零設定、100% 離線的 AI 智慧整理 (Zero-Config Local AI)**
- **免 API Key、免綁卡**：直接呼叫 Chrome 內建 Gemini Nano 模型，按一下按鈕，AI 自動分析語意並將數十個分頁分類到對應的群組中。
- **Hover 智慧摘要**：游標懸停在分頁上 2 秒，AI 自動為您產出一句話重點摘要，找資料不再盲目點擊！
- **極致隱私**：所有運算皆在您的設備端進行，資料絕不上傳第三方伺服器。

**2. 🎨 畫布級的客製化與 Arc-like 美學**
- 深度打磨的玻璃擬物 (Glassmorphism) 材質與絲滑的拖曳微動畫。
- 內建精美主題（Geek, Darcula, Google 等），支援完全自訂的 HSL 主題色與自訂背景圖片，支援 JSON 匯出/匯入社群字型。

**3. 🔗 虛實整合：Linked Tabs**
- 將分頁與書籤的界線徹底模糊。從側邊欄打開書籤時會自動建立連結，雙向同步狀態（例如 F2 重新命名），避免重複開啟相同分頁。

**4. 📰 專注吸收：閱讀清單與輕量 RSS**
- 內建與 Chrome 同步的閱讀清單，並支援自訂 RSS 訂閱——自動定時背景抓取新文章，不需額外安裝肥大的閱讀器擴充功能。

---

**✨ 核心強大功能清單：**

✅ **垂直分頁與群組**：一目了然的直式分頁樹狀圖，完整支援 Chrome 內建的分頁群組 (Tab Groups) 展開與收合。
✅ **跨視窗管理**：不只當前視窗，總覽並搜尋全域所有 Chrome 視窗中的分頁。
✅ **專業級搜尋**：多關鍵字過濾、Domain 限定搜尋、即時高亮，瞬間找到目標。
✅ **全程鍵盤導覽**：方向鍵切換、`F2` 重新命名、`Delete` 刪除，讓 Power User 雙手不離鍵盤。

**[系統需求]**
- Hover 摘要與 AI 整理需要啟用 Chrome Flags (`optimization-guide-on-device-model` 與 `prompt-api-for-gemini-nano`)，詳見套件內「AI 設定」指南。
- 支援 Windows / macOS / Linux。
