當前最新版本 - 1.8.0
✨ 新功能 (New Features)
- **強大的搜尋增強**：
    - **多關鍵字搜尋**：支援使用空白分隔多個關鍵字，讓搜尋更靈活（OR 邏輯）。
    - **URL Domain 搜尋**：現在可以搜尋分頁和書籤的網址 Domain。
    - **智慧顯示**：當搜尋匹配到 URL 時，會自動在標題下方顯示匹配的 Domain 資訊。
    - **即時高亮**：搜尋結果中的關鍵字會被高亮顯示，並支援所有主題配色。
    - **結果計數**：即時顯示搜尋到的分頁與書籤數量。
    - **清除按鈕**：新增一鍵清除搜尋內容的按鈕。

🚀 改善與錯誤修復 (Improvements & Bug Fixes)
- **多語系支援**：為所有 14 種支援語言補齊了搜尋功能的翻譯。
- **介面優化**：修正了搜尋結果中 Icon 與標題的對齊問題，並優化了 Domain 顯示的間距。
- **效能優化**：加入 Debounce 機制，提升搜尋時的流暢度。

厌倦了在数十个杂乱无章的标签页中迷失方向吗？渴望 Arc 浏览器那般流畅优雅的标签页管理体验，却又离不开 Chrome？你的愿望，现在实现！🚀

Arc 风格侧边栏将 Chrome 彻底改造，引入一个集标签页、标签页组、书签于一体的统一侧边面板。告别以往在多个窗口和书签管理器间来回切换的混乱，重新掌控你的浏览器，大幅提升生产力与专注力。

核心亮点

🗂️ 统一工作空间
在单一界面中无缝管理所有标签页与书签。一个面板，掌控一切，视野无比清晰。

🤏 直观的拖放体验
自由拖放标签页、标签页组、书签和文件夹，随心所欲地重组你的工作区。你的空间，你做主。

✨ 拖拽即存为书签
灵感闪现？只需将标签页轻轻一拖，放入书签区即可永久收藏。

📂 悬停自动展开文件夹
整理书签时，不再需要繁琐的点击。只需将项目悬停在折叠的文件夹上，它便会为你自动展开。

⚡️ 智能悬停操作
一个干净、无干扰的界面，只有当你将鼠标悬停其上时，才会优雅地显示关闭、编辑或删除等快捷操作。

🧠 记忆您的检视习惯
侧边栏会智能记住你展开的书签文件夹状态。每次打开，都是你最熟悉的布局。

🎨 完美整合原生标签页组
完美兼容 Chrome 原生的标签页组！组名称、颜色都将被精美呈现，并可随时折叠与展开。

🔍 即时模糊搜索
输入关键字，即可在数百个标签页和书签中进行闪电般的模糊搜索。眨眼之间，找到所需。

💻 为专注而生的主题
享受一个时尚、高对比度的深色主题，搭配画龙点睛的绿色，为长时间的专注工作与舒适阅读而生。

⌨️ 高效率快捷键
- 开关侧边栏: Cmd/Ctrl + I
- 右侧新增标签页: Opt/Alt + T
