## 🔥 核心亮点

### 🔗 独家创新：Linked Tabs (标签页与书签的完美同步)
这是我们最强大的功能！当您从侧边栏打开书签时，我们会自动建立 **“链接 (Link)”**。
- **避免太多重复标签页**：再次点击该书签的 link 图标，会显示已通过该书签开启的标签页，避免无脑新增大量相同标签页造成性能上损耗。
- **双向同步**：标签页关闭时，书签状态自动更新；书签删除时，关联标签页智能处理。
- **视觉反馈**：书签旁会显示精致的链接图标，让您一眼掌握哪些书签正在使用中。

### ⚡️ 智能渲染 (Smart Rendering)
书签多到爆炸？别担心！
- **动态渲染技术**：采用高效的 Dynamic Rendering 机制，弃用传统的 Virtual Scrolling，在保持性能的同时大幅提升兼容性。
- **流畅体验**：即使拥有大量书签，侧边栏依然反应灵敏。

### 🪟 跨窗口管理 (Cross-Window Management)
- **总览所有窗口**：不只是当前窗口，您可以在侧边栏下方看到所有开启中的 Chrome 窗口与标签页。
- **跨窗口搜索**：搜索功能全面支持跨窗口检索，快速切换到任何已开启的标签页。
### 🔍 专业级搜索
不只是搜索，是“瞬间找到”。
- **多关键字过滤**：支持空格分隔 (例如：“google docs work”)，精准定位目标。
- **Domain 搜索**：直接输入域名 (如 `github.com`)，快速筛选特定来源的标签页与书签。
- **智能高亮**：实时标示匹配关键字，视觉焦点一目了然。

### 🗂️ 统一工作空间
- **垂直标签页**：让标题完整显示，不再被压缩成小图标。
- **原生组支持**：完美整合 Chrome 标签页组 (Tab Groups)，颜色、名称同步显示。
- **自定义窗口命名**：可为每个窗口设定专属名称（如“工作”、“娱乐”），多窗口管理更清晰。
- **拖拽管理**：直观的 Drag & Drop，轻松在标签页、组、书签文件夹之间移动项目。
- **拖拽即存**：将标签页拖入书签区，立即收藏；将书签拖入标签页区，立即开启。
- **阅读清单 & RSS**：您的个人文章收藏中心。与 Chrome 原生阅读清单同步，订阅 RSS 频道，并通过智能去重与排序功能统一管理。

### 🎨 极致美学与个人化 (Premium Design & Customization)
- **自定义背景图片**：打造专属工作空间！支持上传图片或输入 URL，并可精确调整 **不透明度** 与 **模糊度**，让侧边栏完美融入您的风格。
- **全新设置面板 UI**：现代化的 **折叠式（Accordion）布局**，让设置项分类更清晰，操作更直观、更简洁。
- **自定义主题颜色**：打破预设框架！现在您可以自由调整背景色、强调色与文字颜色，完全掌控视觉呈现。
- **专注模式**：精緻的深色主题，经过仔细调整的对比度，减轻眼部负担。
- **自动展开**：在拖曳项目时悬停于文件夹上，会自动展开路径。
- **智能悬停**：功能按钮仅在需要时出现，保持界面简洁且无干扰。

## ⌨️ 完整键盘导航与快捷键 (Full Keyboard Navigation)
- **原生体验**：使用 `向上`/`向下` 鍵在标签页与书籤之间流畅切换。
- **微互动**：使用 `向左`/`向右` 鍵进行巡览并触发内部按钮（如关闭、加入群组）。
- **高效率快捷键**：
    - `F2`：快速重命名任何标签页或书签。
    - `Delete`：一键删除不需要的项目。
    - `Cmd/Ctrl + I`：切换侧边栏。
    - `Opt/Alt + T`：在当前标签页旁创建新标签页。
- **搜索整合**：在列表顶部按 `上` 即可聚焦搜索框；在搜索框按 `下` 即可跳转至结果。
- **聚焦秘笈**：侧边栏开启后，只需按下任何方向键即可自动取得焦点并开始导航。
