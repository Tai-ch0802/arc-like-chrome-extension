🔒 **100% ローカル** · 🔑 **API キー不要** · 🤖 **内蔵 Gemini Nano AI**

- **AI タブクリーンアップ提案**: 🧹 ボタンで AI が閉じてよいタブを提案。

## 🔥 主な機能

### 🔗 独自の革新：リンクされたタブ (Linked Tabs)
これが私たちの最も強力な機能です！サイドバーからブックマークを開くと、自動的に**「リンク」**が作成されます。
- **タブの乱雑さを回避**: ブックマークの横にあるリンクアイコンをクリックすると、そのブックマークから開かれたすべてのタブが表示され、重複して開くのを防ぎ、システムリソースを節約できます。
- **双方向同期**：タブを閉じると、ブックマークの状態が自動的に更新されます。ブックマークを削除すると、リンクされたタブはインテリジェントに処理されます。
- **視覚的フィードバック**：ブックマークの横に洗練されたリンクアイコンが表示され、どれが現在アクティブかが一目でわかります。

### ⚡️ スマートレンダリング (Smart Rendering)
数千のブックマークがあっても問題ありません！
- **ダイナミックレンダリング**：仮想スクロールから効率的なダイナミックレンダリングメカニズムに切り替え、スムーズなパフォーマンスと互換性を向上させました。
- **スムーズな体験**：大量のブックマークライブラリでもラグなく操作できます。

### 🪟 クロスウィンドウ管理
- **ウィンドウ一覧**：現在のウィンドウだけでなく、開いているすべてのChromeウィンドウのタブをサイドバーで直接確認できます。
- **グローバル検索**：検索結果にはすべてのウィンドウのタブが含まれ、セッション全体で瞬時に移動できます。
### 🔍 プロ仕様の検索
ただ探すのではなく、瞬時に見つけます。
- **マルチキーワードフィルタリング**：スペース区切りのキーワード（例：「google docs 仕事」）をサポートし、正確なターゲティングを可能にします。
- **ドメイン検索**：ドメイン（`github.com`など）を入力すると、特定のソースからのタブやブックマークを瞬時にフィルタリングできます。
- **スマートハイライト**：一致したキーワードをリアルタイムでハイライトし、視覚的な焦点を明確に保ちます。

### 🗂️ 統合ワークスペース
- **垂直タブ**：ページタイトル全体を表示し、小さなアイコンに圧縮されることはもうありません。
- **ネイティブグループサポート**：Chromeタブグループと完全に統合され、色と名前が同期されます。
- **ウィンドウのカスタム命名**：ウィンドウにカスタム名（例：「仕事」、「個人」）を割り当てて、コンテキストを明確にします。
- **ドラッグ＆ドロップ**：直感的な管理—タブ、グループ、ブックマークフォルダ間でアイテムを簡単に移動できます。
- **ドラッグして保存**：タブをブックマークエリアにドラッグして瞬時に保存。ブックマークをタブエリアにドラッグして開きます。
- **リーディングリスト & RSS**: あなた専用の記事キュレーションハブ。Chrome ネイティブのリーディングリストと同期し、RSS フィードを購読して、スマートな重複排除と並べ替え機能で一元管理できます。

### 🎨 プレミアムデザインとカスタマイズ (Premium Design & Customization)
- **カスタム背景画像**：自分だけのワークスペースを！アップロードまたは URL から背景を設定可能。**不透明度**や**ぼかし**を微調整して、理想のスタイルに。
- **刷新された設定 UI**：モダンで整理された**アコーディオン形式のレイアウト**を導入。設定の変更がこれまで以上にスムーズになります。
- **カスタムテーマカラー**：プリセットの枠を超えて。メインの背景色、アクセントカラー、テキストの色を自由にコントロール。
- **フォーカスモード**：目の疲れを軽減するために慎重に調整されたコントントラストを備えた、洗練されたダークテーマ。
- **自動展開**：アイテムをドラッグ中にフォルダの上にホバーすると、自動的にパスが展開されます。
- **スマートホバー**：アクションボタンは必要なときにだけ表示され、インターフェースをクリーンで邪魔にならない状態に保ちます。

## ⌨️ フルキーボードナビゲーションとショートカット
- **ネイティブ体験**：`上`/`下` キーを使用して、タブとブックマークの間をシームレスに移動。
- **マイクロインタラクション**：`左`/`右` キーを使用してナビゲートし、内部ボタン（閉じる、グループに追加など）をトリガー。
- **生産性向上のためのショートカット**：
    - `F2`：タブやブックマークを即座に名前変更。
    - `Delete`：不要なアイテムを素早く削除。
    - `Cmd/Ctrl + I`：サイドバーの切り替え。
    - `Opt/Alt + T`：現在のタブの横に新しいタブを作成。
- **検索の統合**：リストの先頭で `上` を押すと検索バーにフォーカス。検索バーで `下` を押すと結果にジャンプ。
- **フォーカスのヒント**：サイドバーが開いたら、矢印キーを押すだけで自動的にフォーカスを取得し、ナビゲーションを開始できます。
