# Arc-like Vertical Tabs & Bookmarks - Chrome को एक उत्पादकता पावरहाउस में बदलें · Chrome के लिए आपका Knowledge Workspace

🔒 **Local by default** · 🔑 **No API key needed (BYO optional)** · 🤖 **Built-in Gemini Nano AI**

Chrome के native vertical tabs (2025) सिर्फ tabs को vertical दिखाते हैं। **यह एक पूरा knowledge workspace है**: tabs + bookmarks + reading list एक साथ, AI (group auto-name, tab cleanup, hover summary, natural-language search), Workspaces एक click में hibernate-restore और devices के बीच sync, ⌘K Command Palette, Bookmark Tools (tags, dedupe, dead-link) —— डिफ़ॉल्ट रूप से device पर (कोई API key ज़रूरी नहीं), साथ में वैकल्पिक bring-your-own-key cloud AI (Gemini API / Claude / OpenAI-compatible / Ollama)।

क्या आप दर्जनों अव्यवस्थित टैब में खो जाने से थक गए हैं? क्या आप Arc ब्राउज़र के स्लीक, वर्टिकल टैब अनुभव की लालसा रखते हैं लेकिन Chrome इकोसिस्टम को नहीं छोड़ सकते?
**Arc-like Vertical Tabs & Bookmarks** आपका अंतिम उत्तर है! 🚀

हमने Chrome की पूरी तरह से फिर से कल्पना की है, एक आधुनिक साइड पैनल पेश किया है जो **वर्टिकल टैब**, **टैब समूह** और **बुकमार्क प्रबंधन** को एकीकृत करता है। पारंपरिक क्षैतिज टैब बार को अलविदा कहें और अपने ब्राउज़र पर नियंत्रण वापस पाएं। पहले कभी न देखे गए फोकस और दक्षता का अनुभव करें।

## 🔥 मुख्य विशेषताएं

### 🔗 विशेष नवाचार: लिंक्ड टैब (Linked Tabs)
यह हमारी सबसे शक्तिशाली विशेषता है! जब आप साइडबार से कोई बुकमार्क खोलते हैं, तो हम स्वचालित रूप से एक **"लिंक"** बनाते हैं।
- **टैब अव्यवस्था से बचें**: किसी बुकमार्क के बगल में लिंक आइकन पर क्लिक करके उससे खोले गए सभी टैब देखें, जिससे आपको डुप्लिकेट खोलने से बचने और सिस्टम संसाधनों को बचाने में मदद मिलती है।
- **दो-तरफा सिंक**: जब कोई टैब बंद हो जाता है, तो बुकमार्क स्थिति स्वचालित रूप से अपडेट हो जाती है; जब कोई बुकमार्क हटा दिया जाता है, तो लिंक किए गए टैब को समझदारी से संभाला जाता है।
- **दृश्य प्रतिक्रिया**: बुकमार्क के बगल में एक परिष्कृत लिंक आइकन दिखाई देता है, जिससे आपको एक नज़र में पता चल जाता है कि वर्तमान में कौन से सक्रिय हैं।

### 🔍 पेशेवर ग्रेड खोज
सिर्फ खोजें नहीं—तुरंत पाएं।
- **मल्टी-कीवर्ड फ़िल्टरिंग**: सटीक लक्ष्यीकरण के लिए स्पेस-सेपरेटेड कीवर्ड (जैसे "google docs work") का समर्थन करता है।
- **डोमेन खोज**: विशिष्ट स्रोतों से टैब और बुकमार्क को तुरंत फ़िल्टर करने के लिए एक डोमेन (जैसे `github.com`) टाइप करें।
- **स्मार्ट हाइलाइटिंग**: मेल खाने वाले कीवर्ड की रीयल-टाइम हाइलाइटिंग आपके दृश्य फोकस को स्पष्ट रखती है।

### 🗂️ एकीकृत कार्यक्षेत्र
- **वर्टिकल टैब**: पूर्ण पृष्ठ शीर्षक देखें, अब छोटे आइकन में संकुचित नहीं।
- **नेटिव ग्रुप सपोर्ट**: Chrome टैब समूहों के साथ पूरी तरह से एकीकृत होता है, रंगों और नामों को सिंक करता है।
- **ड्रैग एंड ड्रॉप**: सहज प्रबंधन—टैब, समूहों और बुकमार्क फ़ोल्डरों के बीच आइटम को आसानी से ले जाएं।
- **सहेजने के लिए खींचें**: इसे तुरंत सहेजने के लिए बुकमาร์क क्षेत्र में एक टैब खींचें; इसे खोलने के लिए टैब क्षेत्र में एक बुकमार्क खींचें।
- **रीडिंग लिस्ट और RSS**: आर्टिकल क्यूरेशन के लिए आपका व्यक्तिगत केंद्र। नेटिव Chrome रीडिंग लिस्ट के साथ सिंक करें, RSS फीड सब्सक्राइब करें, और स्मार्ट डिडुप्लीकेशन और सॉर्टिंग के साथ सब कुछ एक ही स्थान पर प्रबंधित करें।
- **कस्टम विंडो नामकरण**: स्पष्ट संदर्भ के लिए अपनी खिड़कियों (जैसे, "कार्य", "व्यक्तिगत") को कस्टम नाम दें।

### 🎨 प्रीमियम डिज़ाइन और अनुकूलन
- **कस्टम बैकग्राउंड इमेज**: अपने वर्कस्पेस को निजीकृत करें! अपनी खुद की इमेज अपलोड करें या URL का उपयोग करें। सही लुक के लिए इसे समायोज्य **पारदर्शिता** और **धुंधलापन** के साथ ट्यून करें।
- **नया सेटिंग्स UI**: आधुनिक और व्यवस्थित सेटिंग्स पैनल का आनंद लें, जिसमें एक नया **एकॉर्डियन लेआउट** है, जिससे आपके साइडबार को कॉन्फ़िगर करना पहले से कहीं अधिक आसान हो गया है।
- **कस्टम थीम रंग**: प्रीसेट से आगे बढ़ें। अब आपके पास मुख्य पृष्ठभूमि, एक्सेंट और टेक्स्ट रंगों पर पूर्ण नियंत्रण है।
- **फोकस मोड**: आंखों के तनाव को कम करने के लिए सावधानीपूर्वक ट्यून किए गए कंट्रास्ट के साथ एक चिकना डार्क थीम।
- **ऑटो-एक्सपैंड**: पथ को स्वचालित रूप से विस्तारित करने के लिए आइटम खींचते समय फ़ोल्डरों पर होवर करें।
- **स्मार्ट होवर**: एक्शन बटन केवल आवश्यकता पड़ने पर दिखाई देते हैं, जिससे इंटरफ़ेस साफ और व्याकुलता-मुक्त रहता है।

## ⌨️ पूर्ण कीबोर्ड नेविगेशन और शॉर्टकट
- **नेटिव अनुभव**: टैब और बुकमार्क के बीच निर्बाध रूप से नेविगेट करने के लिए `Arrow Up`/`Arrow Down` कुंजियों का उपयोग करें।
- **माइक्रो-इंटरैक्शन**: नेविगेट करने और आंतरिक बटन (जैसे क्लोज, ऐड टू ग्रुप) को ट्रिगर करने के लिए `Left`/`Right` का उपयोग करें।
- **उत्पादकता शॉर्टकट**:
    - `F2`: किसी भी टैब या बुकमार्क का तुरंत नाम बदलें।
    - `Delete`: अवांछित आइटम को जल्दी से हटा दें।
    - `Cmd/Ctrl + I`: साइडबार टॉगल करें।
    - `Opt/Alt + T`: वर्तमान के बगल में एक नया टैब बनाएं।
- **खोज एकीकरण**: खोज बार पर ध्यान केंद्रित करने के लिए सूची के शीर्ष पर `Up` दबाएं; परिणामों पर जाने के लिए खोज बार में `Down` दबाएं।
- **फोकस टिप**: एक बार साइडबार खुला होने पर, स्वचालित रूप से फोकस लेने और नेविगेशन शुरू करने के लिए बस किसी भी तीर कुंजी को दबाएं।
