## 🔥 Ключевые особенности

### 🔗 Эксклюзивная инновация: Связанные вкладки (Linked Tabs)
Это наша самая мощная функция! Когда вы открываете закладку с боковой панели, мы автоматически создаем **"Связь"**.
- **Избегайте беспорядка во вкладках**: Нажмите на значок ссылки рядом с закладкой, чтобы увидеть все открытые из нее вкладки. Это поможет избежать дубликатов и сэкономить системные ресурсы.
- **Двусторонняя синхронизация**: Когда вкладка закрывается, статус закладки обновляется автоматически; когда закладка удаляется, связанная вкладка обрабатывается интеллектуально.
- **Визуальная обратная связь**: Рядом с закладками появляется изящный значок связи, позволяющий с первого взгляда понять, какие из них активны в данный момент.

### ⚡️ Smart Rendering
Have thousands of bookmarks? No problem!
- **Dynamic Rendering**: Switched from Virtual Scrolling to an efficient Dynamic Rendering mechanism, ensuring smooth performance with better compatibility.
- **Smooth Experience**: Navigate through large bookmark libraries effortlessly without lag.

### 🪟 Cross-Window Management
- **Window Overview**: View tabs from all open Chrome windows directly in the sidebar, not just the current one.
- **Global Search**: Search results include tabs from all windows, allowing for instant navigation across your entire session.
### 🔍 Поиск профессионального уровня
Не просто ищите — находите мгновенно.
- **Фильтрация по нескольким ключевым словам**: Поддерживает ключевые слова, разделенные пробелами (например, "google docs работа"), для точного поиска.
- **Поиск по домену**: Введите домен (например, `github.com`), чтобы мгновенно отфильтровать вкладки и закладки из определенных источников.
- **Умная подсветка**: Подсветка совпадающих ключевых слов в реальном времени сохраняет ваш визуальный фокус четким.

### 🗂️ Единое рабочее пространство
- **Вертикальные вкладки**: Просматривайте полные заголовки страниц, больше не сжатые в крошечные значки.
- **Нативная поддержка групп**: Идеально интегрируется с группами вкладок Chrome, синхронизируя цвета и названия.
- **Drag & Drop**: Интуитивное управление — легко перемещайте элементы между вкладками, группами и папками закладок.
- **Перетащите, чтобы сохранить**: Перетащите вкладку в область закладок, чтобы мгновенно сохранить ее; перетащите закладку в область вкладок, чтобы открыть ее.
- **Список для чтения и RSS**: Ваш личный центр для сбора статей. Синхронизируйте с нативным списком для чтения Chrome, подписывайтесь на RSS-каналы и управляйте всем в одном месте с умным удалением дубликатов и сортировкой.
- **Пользовательские имена окон**: Присваивайте окнам собственные имена (например, «Работа», «Личное») для лучшей организации.

### 🎨 Премиальный дизайн и кастомизация
- **Пользовательское фоновое изображение**: Персонализируйте свое рабочее пространство! Загрузите свое изображение или используйте URL. Настройте **прозрачность** и **размытие** для идеального вида.

- **Режим фокуса**: Элегантная темная тема с тщательно подобранным контрастом для снижения нагрузки на глаза.
- **Автоматическое развертывание**: Наведите курсор на папки при перетаскивании элементов, чтобы автоматически развернуть путь.

## ⌨️ Полная навигация с клавиатуры и горячие клавиши
- **Нативный опыт**: Используйте клавиши `Стрелка вверх`/`вниз` для плавного переключения между вкладками и закладками.
- **Микровзаимодействия**: Используйте `Влево`/`Вправо` для навигации и активации внутренних кнопок (закрыть, добавить в группу).
- **Горячие клавиши для продуктивности**:
    - `F2`: Мгновенное переименование любой вкладки или закладки.
    - `Delete`: Быстрое удаление ненужных элементов.
    - `Cmd/Ctrl + I`: Переключить боковую панель.
    - `Opt/Alt + T`: Создать новую вкладку рядом с текущей.
- **Интеграция с поиском**: Нажмите `Вверх` в начале списка, чтобы сфокусироваться на строке поиска; нажмите `Вниз` в строке поиска, чтобы перейти к результатам.
- **Совет по фокусу**: Как только боковая панель открыта, просто нажмите любую клавишу со стрелкой, чтобы автоматически получить фокус и начать навигацию.
