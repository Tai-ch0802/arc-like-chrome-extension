# แท็บแนวตั้งและบุ๊กมาร์กสไตล์ Arc - เปลี่ยน Chrome ให้เป็นขุมพลังแห่งการทำงาน · พื้นที่ทำงานความรู้ของคุณบน Chrome

🔒 **บนเครื่องเป็นค่าเริ่มต้น** · 🔑 **ไม่ต้องใช้ API key (นำคีย์ของคุณเองมาใช้ได้)** · 🤖 **Gemini Nano AI ในเครื่อง**

Vertical tabs ของ Chrome (2025) แค่แสดงแท็บแนวตั้ง **นี่คือ knowledge workspace ที่สมบูรณ์**: tabs + bookmarks + reading list ในที่เดียว, AI (ตั้งชื่อกลุ่มอัตโนมัติ, cleanup tab, hover summary, natural-language search), Workspaces hibernate-restore คลิกเดียวซิงค์ข้ามอุปกรณ์, ⌘K Command Palette, Bookmark Tools (tags, dedupe, dead-link) — ค่าเริ่มต้นทำงานบนเครื่องโดยไม่ต้องใช้ API key และนำ API key ของคุณเองมาเชื่อมต่อโมเดลคลาวด์ได้ (Gemini API / Claude / OpenAI-compatible / Ollama)

เบื่อกับการหลงทางในแท็บที่ยุ่งเหยิงนับสิบหรือไม่? ปรารถนาประสบการณ์แท็บแนวตั้งที่โฉบเฉี่ยวของเบราว์เซอร์ Arc แต่ไม่สามารถออกจากระบบนิเวศของ Chrome ได้ใช่ไหม?
**Arc-like Vertical Tabs & Bookmarks** คือคำตอบสุดท้ายของคุณ! 🚀

เราได้จินตนาการ Chrome ใหม่ทั้งหมด โดยนำเสนอแผงด้านข้างที่ทันสมัยซึ่งรวม **แท็บแนวตั้ง**, **กลุ่มแท็บ** และ **การจัดการบุ๊กมาร์ก** เข้าด้วยกัน บอกลาแถบแท็บแนวนอนแบบเดิมๆ และทวงคืนการควบคุมเบราว์เซอร์ของคุณ สัมผัสประสบการณ์การโฟกัสและประสิทธิภาพอย่างที่ไม่เคยมีมาก่อน

### 🤖 Knowledge Workspace 2026 (ใหม่)
- **AI Auto Group Naming**: กลุ่มแท็บว่างใหม่ได้ emoji + ป้ายสั้นอัตโนมัติ
- **AI Tab Cleanup Suggestions**: ปุ่ม 🧹, AI บอกว่าแท็บไหนปิดได้
- **AI Hover Summary**: hover บนแท็บ 2 วินาที → สรุปหนึ่งประโยค
- **AI Reading List Summary Memory**: เมื่อเพิ่มแล้ว AI สรุปและบันทึกในเครื่อง อ่านได้แม้ออฟไลน์
- **Ask AI Natural-Language Search (⌘K)**: ประโยคเดียวค้น tabs / bookmarks / reading list
- **Workspaces**: ชุดแท็บที่ตั้งชื่อ, hibernate-restore, metadata ซิงค์ข้ามอุปกรณ์
- **Command Palette (⌘K / Ctrl+K)**: ค้นหา + การกระทำในที่เดียว
- **Bookmark Tools**: 🛠️ tags / dedupe / dead-link cleanup
- **ในเครื่องเป็นค่าเริ่มต้น คลาวด์เป็นตัวเลือก**: AI ทำงานบน Gemini Nano ที่ Chrome ติดมาให้ตั้งแต่แกะกล่อง — ข้อมูลไม่ออกจากเครื่องของคุณ และเลือกเชื่อมต่อ API key ของคุณเองได้ (Gemini API / Claude / OpenAI-compatible / Ollama)

## 🔥 คุณสมบัติหลัก

### 🔗 นวัตกรรมเฉพาะ: แท็บที่เชื่อมโยง (Linked Tabs)
นี่คือคุณสมบัติที่ทรงพลังที่สุดของเรา! เมื่อคุณเปิดบุ๊กมาร์กจากแถบด้านข้าง เราจะสร้าง **"ลิงก์"** โดยอัตโนมัติ
- **หลีกเลี่ยงแท็บที่รก**: คลิกไอคอนลิงก์ถัดจากบุ๊กมาร์กเพื่อดูแท็บทั้งหมดที่เปิดจากบุ๊กมาร์กนั้น ซึ่งช่วยให้คุณหลีกเลี่ยงการเปิดซ้ำและประหยัดทรัพยากรระบบ
- **การซิงค์สองทาง**: เมื่อแท็บถูกปิด สถานะบุ๊กมาร์กจะอัปเดตโดยอัตโนมัติ; เมื่อบุ๊กมาร์กถูกลบ แท็บที่เชื่อมโยงจะได้รับการจัดการอย่างชาญฉลาด
- **การตอบสนองด้วยภาพ**: ไอคอนลิงก์ที่ประณีตจะปรากฏขึ้นข้างบุ๊กมาร์ก ช่วยให้คุณทราบได้ทันทีว่าอันไหนกำลังใช้งานอยู่

### ⚡️ Smart Rendering
Have thousands of bookmarks? No problem!
- **Dynamic Rendering**: Switched from Virtual Scrolling to an efficient Dynamic Rendering mechanism, ensuring smooth performance with better compatibility.
- **Smooth Experience**: Navigate through large bookmark libraries effortlessly without lag.

### 🪟 Cross-Window Management
- **Window Overview**: View tabs from all open Chrome windows directly in the sidebar, not just the current one.
- **Global Search**: Search results include tabs from all windows, allowing for instant navigation across your entire session.
### 🔍 การค้นหาระดับมืออาชีพ
อย่าแค่ค้นหา—แต่จงค้นพบในทันที
- **การกรองหลายคำหลัก**: รองรับคำหลักที่คั่นด้วยช่องว่าง (เช่น "google docs งาน") เพื่อการกำหนดเป้าหมายที่แม่นยำ
- **การค้นหาโดเมน**: พิมพ์โดเมน (เช่น `github.com`) เพื่อกรองแท็บและบุ๊กมาร์กจากแหล่งที่มาเฉพาะในทันที
- **การไฮไลท์อัจฉริยะ**: การไฮไลท์คำหลักที่ตรงกันแบบเรียลไทม์ช่วยให้การโฟกัสทางสายตาของคุณชัดเจน

### 🗂️ พื้นที่ทำงานแบบครบวงจร
- **แท็บแนวตั้ง**: ดูชื่อหน้าเต็ม ไม่ถูกบีบอัดเป็นไอคอนเล็กๆ อีกต่อไป
- **รองรับกลุ่มแบบเนทีฟ**: ผสานรวมกับกลุ่มแท็บ Chrome ได้อย่างสมบูรณ์แบบ ซิงค์สีและชื่อ
- **ลากและวาง**: การจัดการที่ใช้งานง่าย—ย้ายรายการระหว่างแท็บ กลุ่ม และโฟลเดอร์บุ๊กมาร์กได้อย่างง่ายดาย
- **ลากเพื่อบันทึก**: ลากแท็บไปที่พื้นที่บุ๊กมาร์กเพื่อบันทึกทันที; ลากบุ๊กมาร์กไปที่พื้นที่แท็บเพื่อเปิด
- **รายการอ่านและ RSS**: ศูนย์รวมการคัดสรรบทความส่วนตัวของคุณ ซิงค์กับรายการอ่านดั้งเดิมของ Chrome สมัครรับฟีด RSS และจัดการทุกอย่างในที่เดียวด้วยการลบรายการซ้ำอัจฉริยะและการจัดเรียง
- **การตั้งชื่อหน้าต่างแบบกำหนดเอง**: ตั้งชื่อหน้าต่างของคุณเอง (เช่น "งาน", "ส่วนตัว") เพื่อบริบทที่ชัดเจนยิ่งขึ้น

### 🎨 การออกแบบระดับพรีเมียมและการปรับแต่ง
- **กำหนดรูปภาพพื้นหลังเอง**: ปรับแต่งพื้นที่ทำงานของคุณ! อัปโหลดรูปภาพของคุณเองหรือใช้ URL ปรับจูนด้วย**ความทึบ**และ**ความเบลอ**ที่ปรับได้เพื่อลุคที่สมบูรณ์แบบ
- **UI การตั้งค่าโฉมใหม่**: เพลิดเพลินกับแผงการตั้งค่าที่ทันสมัยและเป็นระเบียบด้วยเลย์เอาต์แบบ **Accordion** ใหม่ ทำให้การกำหนดค่าแถบด้านข้างของคุณง่ายกว่าที่เคย
- **ปรับแต่งสีธีมเอง**: ก้าวข้ามขีดจำกัดของค่าเริ่มต้น ตอนนี้คุณสามารถควบคุมสีพื้นหลังหลัก สีเน้น และสีข้อความได้อย่างเต็มที่
- **โหมดโฟกัส**: ธีมมืดที่โฉบเฉี่ยวพร้อมคอนทราสต์ที่ปรับแต่งมาอย่างดีเพื่อลดอาการเมื่อยล้าของดวงตา
- **ขยายอัตโนมัติ**: วางเมาส์เหนือโฟลเดอร์ขณะลากรายการเพื่อขยายเส้นทางโดยอัตโนมัติ
- **Smart Hover**: ปุ่มดำเนินการจะปรากฏขึ้นเมื่อจำเป็นเท่านั้น เพื่อให้อินเทอร์เฟซสะอาดตาและไม่รบกวนสมาธิ

## ⌨️ การนำทางด้วยคีย์บอร์ดที่สมบูรณ์และทางลัด
- **ประสบการณ์ดั้งเดิม**: ใช้ปุ่ม `ลูกศรขึ้น`/`ลง` เพื่อนำทางระหว่างแท็บและบุ๊กมาร์กได้อย่างราบรื่น
- **ไมโครอินเทอร์แอกชัน**: ใช้ `ซ้าย`/`ขวา` เพื่อนำทางและเรียกใช้ปุ่มภายใน (เช่น ปิด, เพิ่มในกลุ่ม)
- **ทางลัดเพิ่มประสิทธิภาพ**:
    - `F2`: เปลี่ยนชื่อแท็บหรือบุ๊กมาร์กได้ทันที
    - `Delete`: ลบรายการที่ไม่ต้องการได้อย่างรวดเร็ว
    - `Cmd/Ctrl + I`: เปิด/ปิดแถบด้านข้าง
    - `Opt/Alt + T`: สร้างแท็บใหม่ถัดจากแท็บปัจจุบัน
- **การรวมการค้นหา**: กด `ขึ้น` ที่ด้านบนของรายการเพื่อโฟกัสแถบค้นหา กด `ลง` ในแถบค้นหาเพื่อกระโดดไปยังผลลัพธ์
- **เคล็ดลับการโฟกัส**: เมื่อเปิดแถบด้านข้างแล้ว เพียงกดปุ่มลูกศรใดก็ได้เพื่อดึงโฟกัสโดยอัตโนมัติและเริ่มการนำทาง
